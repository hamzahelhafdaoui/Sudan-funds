INSTRUCTIONS

place these files into your repo

/lib/db.ts
/lib/gofundme.ts
/lib/stats.ts
/app/api/sync/route.ts
/app/api/stats/route.ts
/app/dashboard/page.tsx

install dependency
pnpm add better-sqlite3
or
npm install better-sqlite3

run locally
pnpm dev
import a campaign by visiting
http://localhost:3000/api/sync?url=https://www.gofundme.com/f/YOUR-SLUG
repeat per campaign then open
http://localhost:3000/dashboard

set up a cron hitting /api/sync for each url every 10 minutes to keep donations fresh
