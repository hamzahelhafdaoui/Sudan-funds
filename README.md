# SudanFunds Website

A platform connecting donors with verified Sudanese campaigns.

## About

SudanFunds is a website that helps connect donors worldwide with verified campaigns supporting Sudanese communities in need. Every campaign is thoroughly vetted to ensure transparency and accountability.

## Deployment

This project is deployed on Vercel.

## Development

To run this project locally:

\`\`\`bash
npm install
npm run dev
\`\`\`

Open [http://localhost:3000](http://localhost:3000) to view it in your browser.

## Tech Stack

- Next.js 16
- React 19
- TypeScript
- Tailwind CSS
- shadcn/ui components
